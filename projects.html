Pokemon Collection
Java

Have you ever played Pokemon Go? This program provides users with features within the game
such as store and release Pokemon from their collection, appraise their owned Pokemon to check
stats and check counters for that specific type of Pokemon. In case you happened to be in the
middle of a Pokemon battle, the program also shows you information about your opponent’s Pokemons
for you to strategize your battle!

Discord Bot
Python

A very simple Discord both that I created for the purpose to organize meetings, keep track of meeting time
and send out notifications to potential attendees prior to the meeting. The bot was originally designed for
the Discord channel of my student-run club called New Taiwanese Generations.

Baseball Game Simulator
Java

This baseball game simulator simulates games between two teams made up of greatest players post-2000 including
Clayton Kershaw, Randy Johnson, Derek Jeter, Alex Rodriguez… and many more. Users can also download the simulator
from my GitHub to modify the lineups and give players stats that they deserve.

Chat Bot
Python

A simple chatbot that was originally designed to keep track of my studying time and host a basic conversatios.
The bot uses Machine Learning for the purpose of customization to provide break suggestions based on the knowledge
they have on the user and it is capable of keeping track of how long the user has been studying.

Simple Face Detector
Python

A simple face detector that utilizes the camera on the machine the user is running on to detect people's face.
It runs on the Open CV Library, an open Computer Vision Library that supports Python.
