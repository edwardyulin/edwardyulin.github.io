Pokemon Collection
Java

Have you ever played Pokemon Go? This program provides users with features within the game such as store and release Pokemon from their collection, appraise their owned Pokemon to check stats and check counters for that specific type of Pokemon. In case you happened to be in the middle of a Pokemon battle, the program also shows you information about your opponent’s Pokemons for you to strategize your battle!

Baseball Game Simulator
Java

This baseball game simulator simulates games between two teams made up of greatest players post-2000 including Clayton Kershaw, Randy Johnson, Derek Jeter, Alex Rodriguez… and many more. Users can also download the simulator from my GitHub to modify the lineups and give players stats that they deserve.

Chat Bot
Python

A simple chatbot that was originally designed to keep track of my studying time and host a basic conversatios. The bot uses Machine Learning for the purpose of customization to provide break suggestions based on the knowledge they have on the user and it is capable of keeping track of how long the user has been studying.
